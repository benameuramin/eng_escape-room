====================================================
  THE LOCKED CLASSROOM — Escape Room Game
  English Project by Amin Ben Ameur
====================================================

HOW TO PLAY:
  1. Open index.html in any web browser
  2. Works on phones, tablets, and computers
  3. No internet required after first load (fonts load from Google)

CHALLENGES:
  1. 🎯 The Classic Riddle       — logic & wordplay
  2. 🧠 Memory Master            — memorise 6 items in 5 seconds
  3. 🕵️ Spot the Fake!           — find the false "fact"
  4. 📡 Morse Code Vault         — decode then encode in Morse
  5. 👁️ Find the Mistake         — spot the wrong word in a text

MORSE CODE VAULT (Challenge 4):
  Phase 1 → Decode:  ".-.. . .- .-. -."  =  LEARN
  Phase 2 → Encode:  R(.-.)  E(.)  A(.-)  D(-..)  =  READ

SCORING:
  200 points per correct answer
  Maximum score: 1000 points

TECHNICAL:
  - Single HTML file, no dependencies
  - Mobile-friendly (responsive design)
  - Works offline (except Google Fonts)
